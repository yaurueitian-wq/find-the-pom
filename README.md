# Find the Pomeranian! 🐾

A fun and interactive web-based hide-and-seek game where you search for a hidden Pomeranian (named "薯哥") in progressively challenging images!

## Overview

**Find the Pomeranian!** (找到薯哥!) is a web-based puzzle game featuring 10 levels of increasing difficulty. The game challenges players to locate a hidden Pomeranian dog within complex scenes before time runs out. It's built as a progressive web app (PWA) that works offline and can be installed on mobile devices.

- **10 Challenging Levels**: From easy (park scene) to hard (carnival)
- **60 Seconds Per Level**: Race against the clock to find the hidden pup
- **Score System**: Earn points based on how quickly you find the dog
- **Difficulty Progression**: Each level gets trickier with better camouflage
- **Hints**: A helpful hint appears when you have 20 seconds left
- **Mobile Friendly**: Fully responsive design that works on any device
- **Offline Support**: Play offline as a progressive web app (PWA)

## Game Levels

1. **🌳 Park Afternoon** - Park scene with grass
2. **🏠 Cozy Living Room** - Home interior with carpet
3. **🧸 Dreamy Toy Shop** - Toy store with dolls and stuffed animals
4. **🐱 Cat Café** - Café with cats everywhere
5. **🌸 Colorful Flower Market** - Flower garden market
6. **🏖️ Busy Beach** - Sandy beach scene
7. **📚 Classical Library** - Library with books and sofas
8. **🏮 Night Market** - Evening street food market
9. **☃️ Winter Snow Scene** - Snowy landscape with snowman
10. **🎡 Super Carnival** - Festival with rides and prizes

## Features

- 🎮 **Interactive Gameplay**: Click to locate the hidden Pomeranian
- ⏱️ **Real-time Timer**: Visual countdown with color-coded urgency
- 🎯 **Click Feedback**: Ripple effects and scoring popups
- 🎉 **Celebration Effects**: Confetti and visual feedback when found
- 📊 **Score Breakdown**: Detailed results showing performance per level
- 🎯 **Difficulty Hints**: Star ratings for each level
- 🐕 **Audio Effects**: Bark sounds when you find the dog
- 🌙 **Dark Theme**: Eye-friendly interface with gradient backgrounds
- 📱 **Responsive Design**: Works on desktop, tablet, and mobile
